Introduction to the Attributes of HTML 5


  • All HTML Elements can have Attributes

  • Attributes can provide Additional Information for Elements

  • Attributes are always specified in the start tag

  • Attributes usually come in name/value pairs like name = "value"


    • Use of Attributes

    The Five Attributes are mostly used to give:


  • Width, Height, and Color

  • Form Attributes

  • Inline CSS

  • Class and ID

    • The ID Attribute specifies the unique identity to an HTML element, while the HTML Class attributes specifies one or more class names for an element
