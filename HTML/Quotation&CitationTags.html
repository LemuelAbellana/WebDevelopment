These are the Different Quotation and Citation Tags in HTML 5
br stands for break and is used to break single line in the text

Strong is used to highlight important text by Bolding

P stands for paragraph which is used in creating a text



Em is used to highlight important text by Italicizing the texts

Small is used to show Smaller Text

mark is used to highlight text by coloring the background

Del is used to show the deleted text, this is used when you are in work

ins means inserted text which is used to underline the text

sub stands for subscript which is used to move the text slightly below the text

H 2 O


Sup stands for superscript which is used to move the text slightly above the text

50 2
